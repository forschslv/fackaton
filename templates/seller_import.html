Кабинет импорта товаров из файлов

Загрузите файлы с вашими товарами в формате JSON или CSV для быстрого пакетного добавления во встроенную базу данных.

Поддерживаемые форматы файлов: CSV-таблицы или JSON-массивы.
Все импортируемые позиции моментально валидируются и вносятся во встроенную базу данных SQLite.

Шаблоны форматирования файлов

Структура заголовков CSV

title,category,price,stock,description,image_url,metadata

Например, колонка "metadata" может содержать JSON-строку вида {"цвет":"Черный"}.
Структура массива JSON

[ { "title": "Импортируемый товар A", "category": "Аппаратное обеспечение", "price": 12.99, "stock": 100, "description": "Тестовый импорт", "metadata": { "key_name": "Property_Value" } } ]
